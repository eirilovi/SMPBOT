## Usage

Get your OpenAI API key at https://platform.openai.com/account/api-keys

Create a file named `.env` and add the following:

```
OPENAI_API_KEY=YOURKEY
```

Install dependencies:

```bash
npm Install
```

Run the bot:

```bash
npm start
```
